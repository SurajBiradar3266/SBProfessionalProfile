# linkedInProfileWebsite

Static portfolio site for Suraj Biradar (AUTOSAR Classic & AUTOSAR Adaptive).

## Run locally

Option 1: open `linkedInProfileWebsite/index.html` directly in a browser.

Option 2: run a local web server (recommended for clipboard features):

```powershell
cd linkedInProfileWebsite
python -m http.server 5173
```

Then open `http://localhost:5173`.
